🔄 Algoritma dan Matematika Dasar

Penerapan Logika dalam Algoritma

Algoritma adalah serangkaian langkah logis untuk menyelesaikan masalah. Matematika dasar, khususnya logika, berperan penting dalam:

  • Kondisi dan Percabangan: Menggunakan logika boolean (AND, OR, NOT) untuk membuat keputusan
  • Perulangan: Menerapkan logika untuk menentukan kapan loop harus berhenti
  • Kompleksitas: Menggunakan fungsi matematika untuk menganalisis efisiensi algoritma

Contoh Implementasi

Algoritma Pencarian Binary:

Menggunakan konsep pembagian (fungsi matematika) dan perbandingan (logika) untuk mencari elemen dalam array terurut dengan kompleksitas O(log n).

Algoritma Sorting:

Menerapkan logika perbandingan dan fungsi rekursif untuk mengurutkan data secara efisien.

📊 Struktur Data dan Konsep Matematika

Penerapan Fungsi dalam Struktur Data

Struktur data mengorganisir dan menyimpan data secara efisien menggunakan konsep matematika:

  • Array: Menggunakan fungsi indeks f(i) = base_address + (i × size)
  • Linked List: Menerapkan konsep pointer dan referensi
  • Tree: Menggunakan konsep hierarki dan fungsi rekursif
  • Hash Table: Menerapkan fungsi hash untuk mapping data

Contoh Implementasi

Binary Search Tree:

Menggunakan logika perbandingan (x < parent → kiri, x > parent → kanan) dan fungsi rekursif untuk operasi insert, search, dan delete.

Graph:

Menerapkan konsep relasi matematika dan fungsi untuk merepresentasikan hubungan antar node.

🧠 Logika Matematika dalam Informatika

Logika Proposisional

Dasar pemikiran logis yang digunakan dalam pemrograman:

  • Proposisi: Pernyataan yang bernilai benar atau salah
  • Operator Logika: AND (∧), OR (∨), NOT (¬), IMPLIES (→)
  • Tabel Kebenaran: Menentukan nilai kebenaran dari kombinasi proposisi

Logika Predikat

Logika yang melibatkan variabel dan kuantor:

  • Universal Quantifier (∀): "untuk semua"
  • Existential Quantifier (∃): "ada/terdapat"

Aplikasi dalam Pemrograman

Conditional Statements: if-else menggunakan logika boolean

Loop Conditions: while, for menggunakan evaluasi logika

Database Queries: SQL menggunakan logika predikat

📈 Fungsi Matematika dalam Informatika

Konsep Dasar Fungsi

Fungsi adalah pemetaan dari domain ke range:

  • Domain: Himpunan input yang valid
  • Range: Himpunan output yang mungkin
  • Codomain: Himpunan yang memuat semua kemungkinan output

Jenis-jenis Fungsi

  • Injektif (One-to-One): Setiap elemen domain dipetakan ke elemen range yang berbeda
  • Surjektif (Onto): Setiap elemen range memiliki preimage di domain
  • Bijektif: Fungsi yang injektif dan surjektif

Aplikasi dalam Pemrograman

Hash Functions: Memetakan data ke indeks array

Recursive Functions: Fungsi yang memanggil dirinya sendiri

Lambda Functions: Fungsi anonim untuk operasi sederhana

Big O Notation: Fungsi untuk menganalisis kompleksitas algoritma
